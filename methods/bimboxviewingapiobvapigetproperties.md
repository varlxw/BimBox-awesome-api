> # BIMBox.Viewing.API.OBVApi.getProperties
>
> ---

#### 获取对象的属性。

| 名称 | 类型 | 描述 |
| :--- | :--- | :--- |
| nodeId | Object | nodeId、nodeId\[\]或 dbId、dbId\[\]。为 dbId 时， 默认取第一个模型。 |
| onSuccessCallback | function | 成功取到对象属性后将结果传入此回调函数中。 |
| onErrorCallback | function\(int, string\) | 失败后将调用此回调函数。 |
