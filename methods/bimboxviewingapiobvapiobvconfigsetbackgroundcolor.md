| 名称 | 类型 | 描述 |
| :--- | :--- | :--- |
|   red1 |   Number |   0~255 |
|   green1 |   Number |   0~255 |
|   blue1 |   Number |   0~255 |
|   red2 |   Number |   0~255 |
|   green2 |    Number |   0~255 |
|   blue2 |   Number |   0~255 |

```js
OBVApi.obvConfig.setBackgroundColor(100, 125, 11, 12, 100, 255)
```
